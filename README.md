# Cancer Dataset Resources

Curated links to public and semi-public datasets, registries, terminology systems, and APIs relevant to cancer research, oncology analytics, real-world data, and FHIR-based applications.

## Scope

This repository does not redistribute restricted datasets. It provides links, descriptions, access notes, and licensing/use constraints.

All sample and test data is synthetically generated.

## Dataset Catalog

| Name | Category | Description | Access | Link |
|---|---|---|---|---|
| SEER | Cancer registry | US cancer incidence, survival, and population statistics | Registration required for some data | https://seer.cancer.gov/ |
| Genomic Data Commons | Cancer genomics | NCI cancer genomics and clinical data platform | Open and controlled access | https://gdc.cancer.gov/ |
| ClinicalTrials.gov | Trials | Public registry of clinical studies | Open | https://clinicaltrials.gov/ |
| Synthea | Synthetic data | Synthetic patient records, including FHIR output | Open | https://synthea.mitre.org/ |
| HL7 FHIR | Standard | Healthcare data exchange standard | Open specification | https://hl7.org/fhir/ |
| mCODE | Oncology FHIR model | Minimal Common Oncology Data Elements | Open specification | https://mcodeinitiative.org/ |
| LOINC | Lab terminology | Laboratory and clinical observation codes | Free registration/license | https://loinc.org/ |
| RxNorm | Drug terminology | Normalized drug names and codes | Open via NLM | https://www.nlm.nih.gov/research/umls/rxnorm/ |
| ICD-10-CM | Diagnosis coding | US clinical modification of ICD-10 | Open | https://www.cms.gov/medicare/coding-billing/icd-10-codes |
| SNOMED CT | Clinical terminology | Comprehensive clinical terminology | License-dependent by country | https://www.snomed.org/ |
